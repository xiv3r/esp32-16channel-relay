# Requirements
- ESP32 30/38P
- DUPONT WIRE
- 8 CHANNEL RELAY

# Installations
- Download the firmware and flash using flash offset `0x0`

# Wifi Key
- SSID: `ESP32_8CH_Relay_Controller`
- PASS: `12345678`

# 8 Channel GPIO Connections 
```
GPIO 15  -------> IN1  Relay1
GPIO 2   -------> IN2  Relay2
GPIO 4   -------> IN3  Relay3
GPIO 5   -------> IN4  Relay4
GPIO 18  -------> IN5  Relay5
GPIO 19  -------> IN6  Relay6
GPIO 3   -------> IN7  Relay7
GPIO 1   -------> IN8  Relay8
```
