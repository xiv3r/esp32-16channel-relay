# Requirements
- ESP8266 NodeMCU V1.0
- 13 F-F DUPONT WIRES
- 5v 11 CHANNEL RELAY

# Installations
> Download the firmware in the release and flash to `0x0`

# Wifi Key
- SSID: `ESP8266_11CH_Relay_Controller`
- PASS: `12345678`

# 16 Channel GPIO Connections
```
ESP8266        11 CHANNEL RELAY

GPIO 16  -------> IN1   Relay1
GPIO 5   -------> IN2   Relay2
GPIO 4   -------> IN3   Relay3
GPIO 0   -------> IN4   Relay4
GPIO 2   -------> IN5   Relay5
GPIO 14  -------> IN6   Relay6
GPIO 12  -------> IN7   Relay7
GPIO 13  -------> IN8   Relay8
GPIO 15  -------> IN9   Relay9
GPIO 3   -------> IN10  Relay10
GPIO 1   -------> IN11  Relay11
```
