# Requirements
- ESP32 30/38P
- DUPONT WIRES
- 5v 8 CHANNEL RELAY

# Installations
> Download the firmware in the release and flash to `0x0`

# Wifi Key
- SSID: `ESP32s3_8CH_Relay_Controller`
- PASS: `12345678`

# 16 Channel GPIO Connections
```
ESP32-S3        8 CHANNEL RELAY

GPIO 1  -------> IN1  Relay1
GPIO 2  -------> IN2  Relay2
GPIO 3  -------> IN3  Relay3
GPIO 4  -------> IN4  Relay4
GPIO 5  -------> IN5  Relay5
GPIO 6  -------> IN6  Relay6
GPIO 7  -------> IN7  Relay7
GPIO 8  -------> IN8  Relay8
```
